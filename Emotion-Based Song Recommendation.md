**Emotion-Based Song Recommendation System using NLP \& Machine Learning**

================================================================================



An intelligent Artificial Intelligence and Machine Learning project that aims to identify the emotions of a person based on the text they provide and then give recommendations of songs that match the emotion detected. This is an example of the application of Natural Language Processing and supervised Machine Learning in the development of a personalized experience.



This is an example of the application of Artificial Intelligence in the interpretation of language and the development of an experience that is more engaging.

**## Working Methodology:**



1\. The user enters text describing their emotional state.

2\. The system preprocesses the input text using NLP techniques such as:

&#x20;  • converting text to lowercase

&#x20;  • removing punctuation

&#x20;  • removing stop words

3\. The cleaned text is converted into numerical form using TF-IDF vectorization.

4\. A supervised Machine Learning model predicts the emotion category.

5\. Based on the predicted emotion, the system recommends a suitable song.

6\. The user can repeat the process multiple times.

**## Dataset Description:**



The dataset contains textual sentences labeled with emotional categories.



Emotion categories used in this project:



* Joy
* Sadness
* Anger
* Love
* Fear
* Surprise



The model learns patterns in text associated with each emotion and uses this knowledge to classify new input text.
